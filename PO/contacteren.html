Uitleg geleende code

Hierin geef ik de uitleg over de geleende code die ik heb gebruikt. Allereerst had ik code geleend van W3schools over de nav-bar. Een nav-bar is een lijst van allemaal links waarbij je de "li" commando gebruikt en de "ul" commando. Ik heb ervoor gekozen om een horizontale-navbar te maken. De commando display: block; houdt in hoe je in staat bent om eigenlijk de opvulling te specificeren. Met andere woorden dingen bewerken als de hoogte, breedte enzo. De commando float: left; houdt in dat je de commando "float" gebruikt om blokelementen naast elkaar te laten zweven.
